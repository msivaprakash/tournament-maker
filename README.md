# Tournament Maker

A Progressive Web App for organizing badminton doubles tournaments with real-time scoring and a TV scoreboard.

## Use It

| | Link |
|---|---|
| **Main App** | https://msivaprakash.github.io/tournament-maker/ |
| **TV Scoreboard** | https://msivaprakash.github.io/tournament-maker/tv.html |

Open the main app on your phone to create and manage tournaments. Open the TV scoreboard on a big screen — it pairs via QR code and shows live scores.

## Features

- Schedule matches across multiple courts and rounds
- Live scoring with multi-device sync
- "My Matches" filter for individual players
- TV display with real-time updates, pairing, and celebration screen
- Player stats, history, undo
- Dark mode, installable as PWA

## Dev

```bash
npm install
npm run dev       # local dev server
npm run build     # production build
npm run deploy    # deploy to GitHub Pages
```
